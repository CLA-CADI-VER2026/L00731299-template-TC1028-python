## ☁️ Cómo abrir el Codespace

1. Ingresa al repositorio de esta actividad en GitHub.

2. Haz clic en el botón verde **`<> Code`**, en la esquina superior derecha.

3. Selecciona la pestaña **Codespaces**.

4. Haz clic en **"Create codespace on main"**.

   > ⏳ La primera vez puede tardar entre 1 y 2 minutos en cargar. Ten paciencia.

5. Se abrirá Visual Studio Code en tu navegador con el entorno listo para trabajar.

6. Verifica que Python esté disponible abriendo la terminal integrada
   (**`Ctrl + J`** o menú **Terminal → New Terminal**) y ejecutando:

   ```bash
   python --version
   ```

   Deberías ver algo como:

   ```
   Python 3.12.x
   ```

---

## ▶️ Cómo ejecutar

1. En el panel izquierdo (Explorador), abre el archivo principal de la actividad:

   ```
   main.py
   ```

2. Lee los comentarios dentro del archivo; ahí encontrarás las instrucciones específicas de cada ejercicio.

3. Escribe tu código en las secciones indicadas.

4. Abre la terminal integrada con **`Ctrl + J`** y ejecuta tu programa:

   ```bash
   python main.py
   ```

5. Revisa la salida en la terminal y corrige cualquier error antes de entregar.

   > 💡 **Tip:** Si el programa no hace lo que esperas, agrega `print()` en distintos puntos para inspeccionar el valor de tus variables.

---

## 📦 Entregable

Para entregar esta actividad sigue estos pasos **dentro del Codespace**:

1. Guarda todos los cambios en tu archivo (**`Ctrl + S`**).

2. Abre la terminal y ejecuta los siguientes comandos en orden:

   ```bash
   git add .
   git commit -m "Entrega actividad [NOMBRE]"
   git push
   ```

3. Verifica en GitHub que tus cambios se hayan subido correctamente:
   - Ve a tu repositorio en `github.com`
   - Confirma que el archivo `main.py` muestra tu código actualizado.

4. Copia la URL de tu repositorio y entrégala en la plataforma del curso.

   > ⚠️ **Importante:** No cierres el Codespace sin hacer `git push`. Los cambios que no se suban no serán considerados en la revisión.

## 🆘 ¿Tienes problemas?

| Situación | Qué hacer |
|-----------|-----------|
| El Codespace tarda mucho en cargar | Recarga la página y vuelve a intentarlo |
| `python: command not found` | Prueba con `python3 --version` |
| Error al hacer `git push` | Verifica que estés dentro de tu propio repositorio |
| No encuentras el archivo `main.py` | Revisa el panel Explorador a la izquierda |
